OCR & Computer Vision

Explora diferentes ejemplos y herramientas

  • Tesseract.js FREE

    Tesseract.js es una librería JavaScript para Reconocimiento Óptico de Caracteres (OCR). Es un port de Tesseract, el motor OCR open-source originalmente desarrollado por HP y mantenido por Google.

  • Azure Computer Vision PAID

    Azure Computer Vision es un servicio en la nube dentro de Microsoft Azure AI. Proporciona algoritmos avanzados para analizar, interpretar y extraer información de imágenes y videos.

  • Dynamsoft PDF417 PAID

    Dynamsoft ofrece SDKs de alto rendimiento para lectura de códigos de barras, captura de documentos y escaneo de identificación. Su solución PDF417 se usa comúnmente para licencias y documentos de identidad.
